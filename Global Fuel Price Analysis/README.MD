## **📘 Global Fuel Price Dynamics — Visualization Project

Data Science | Energy Analytics | Matplotlib | Pandas | EDA

📌 Project Overview

This project analyzes global fuel pricing patterns across multiple countries using advanced data visualization and exploratory data analysis (EDA).
The dataset contains 100+ synthetic but realistic daily fuel price records for:

20 Countries

3 Fuel Types: Petrol, Diesel, CNG

Crude Oil Prices

Inflation Rates

Currency Exchange Rates

Monthly & Yearly Trends

The goal is to understand:

How economic indicators impact fuel pricing

Which countries have higher or lower fuel costs

How different fuel types behave globally

How seasonal and volatility patterns emerge

🎯 Project Objectives

✔ Analyze global fuel price movement
✔ Identify country-wise price variation
✔ Compare Petrol vs Diesel vs CNG
✔ Study volatility & price fluctuations
✔ Understand the effect of inflation & crude oil
✔ Analyze seasonal (monthly) trends
✔ Create corporate-quality visualizations


🧠 Key Insights Delivered

Countries with highest & lowest fuel costs

Impact of crude oil changes on fuel pricing

Correlation between inflation and fuel price

Seasonal behavior using monthly median trends

Volatility analysis using standard deviation

Rolling averages revealing smooth long-term trends
